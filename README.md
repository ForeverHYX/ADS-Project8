### How to run?

If you wanna run the code:
- For any ```.py``` file, click right button of your mouse to run in terminal.
- For ```.c``` file, change input file in ```fscanf```, an then run in terminal with
```bash
g++ xxx.c -o -std=c++11 xxx
./xxx
```
